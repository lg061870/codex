# Codex Freight Marketplace

Codex is an Uber-like freight marketplace connecting stores, drivers, and freight customers. The project aims to make moving goods as simple as requesting a ride, providing tools for drivers and store managers to coordinate shipments quickly and efficiently.

## Key Features

- **Request and offer freight services** between users, drivers, and stores
- **Manage vehicles** and driver availability
- **Store logistics tools** for scheduling pickups and deliveries
- **Order tracking** so customers can see where their shipment is at all times

## Architecture Overview

Codex is organized around a mobile application and a set of backend services.

1. **Mobile App** – Used by customers and drivers to book and manage deliveries
2. **Backend Services** – Provide APIs for order management, routing, and notifications
3. **Store Portal** – Allows store personnel to manage inventory and logistics

These components communicate over secure APIs to handle orders, vehicle assignments, and real-time status updates.
